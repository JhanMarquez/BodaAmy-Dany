DESPLEGAR LA API
================

Tu Function App está en plan "Consumo flexible" (Flex). Ese plan NO admite
despliegue por zip (script deploy-with-profile.js); da Error 404.

OPCIÓN RECOMENDADA: Desplegar con Azure CLI
------------------------------------------
1. Instala Azure CLI en Mac (solo una vez):
   https://learn.microsoft.com/cli/azure/install-azure-cli-macos
   Descarga el .pkg, instálalo, reinicia la Terminal.

2. Inicia sesión en Azure (solo la primera vez):
   az login

3. Desde la carpeta del proyecto:
   cd /Users/bluepeople/Documents/Boda/api
   source ~/.zshrc
   func azure functionapp publish boda-amy-dany

4. Copia la URL que muestre al terminar y pónla en index.html en
   CONFIRMACION_API_URL (añade /RegistroConfirmacion al final).

---

Si en el futuro cambias tu Function App a plan "Consumo" (clásico, no Flex),
entonces sí podrás usar: node deploy-with-profile.js con el perfil .PublishSettings.
